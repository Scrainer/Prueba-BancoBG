# Prueba-BancoBG

El repositorio incluye ambos proyectos del api y frontend pedidos.

Para ambos, instalar dependencias con
`npm install`

La base de datos fue trabajada en MongoDB, se puede usar el seed.js en el backend para poblarla.
`node seed.js`

## Iniciar backend

`node app.js`

## Iniciar frontend

`npm start`
